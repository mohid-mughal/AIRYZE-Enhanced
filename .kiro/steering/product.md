# Product Overview

Airyze AQI Monitor is a real-time air quality monitoring application that helps users track air pollution levels across Pakistan and worldwide.

## Core Purpose

Provide real-time air quality information to help users make informed decisions about outdoor activities and protect their health.

## Key Features

- **Interactive Map**: Click anywhere to get real-time AQI data with auto-location detection
- **Major Cities Dashboard**: Monitor AQI for 8 major Pakistani cities (Karachi, Lahore, Islamabad, Rawalpindi, Peshawar, Quetta, Faisalabad, Multan)
- **Historical Analytics**: 30-day AQI trends with line charts and frequency analysis
- **User Accounts**: Registration and login for personalized alerts
- **Email Alerts**: Daily reports (8:00 AM) and significant change notifications (every 30 minutes)
- **Health Recommendations**: Context-aware advice based on current AQI levels
- **Pollutant Breakdown**: Detailed display of PM2.5, PM10, O3, NO2, SO2, CO, and other components

## AQI Scale

Uses US EPA AQI standard (1-5 scale):
- 1 = Good (Green)
- 2 = Fair (Light Green)
- 3 = Moderate (Yellow)
- 4 = Poor (Orange)
- 5 = Very Poor (Red)

## Data Sources

- Real-time data: OpenWeather Air Pollution API
- Historical data: Open-Meteo Air Quality API
